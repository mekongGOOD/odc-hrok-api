API documentation

The following URLs are supported:
  • /census/1998/communes - list all census results (by commune) for 1998.
  • /census/1998/communes/xxx - get result detail, where xxx is the commune ID.
  • /census/1998/districts - list all census results (by district) for 1998.
  • /census/1998/districts/xxx - get result detail, where xxx is the district ID.
  • /census/1998/provinces - list all census results (by province) for 1998.
  • /census/1998/provinces/xxx - get result detail, where xxx is the province ID.
  • /census/1998/villages - list all census results (by village) for 1998.
  • /census/1998/villages/xxx - get result detail, where xxx is the village ID.
  • /census/2008/communes - list all census results (by commune) for 2008.
  • /census/2008/commune/xxx - get result detail, where xxx is the commune ID.
  • /census/2008/districts - list all census results (by district) for 2008.
  • /census/2008/districts/xxx - get result detail, where xxx is the district ID.
  • /census/2008/provinces - list all census results (by province) for 2008.
  • /census/2008/provinces/xxx - get result detail, where xxx is the province ID.
  • /census/2008/villages - list all census results (by village) for 2008.
  • /census/2008/villages/xxx - get result detail, where xxx is the village ID.

The following paging parameters can be used:

Attribute Name Description Default
page The page number. 0
per_page The number of results to return per page. 10

For example to return the first 100 results for the 1998 census results (by commune) the following query would be used:

http://localhost:9090/census/1998/communes?page=0&per_page=100

To retrieve the next 100 results you would use the following query:

http://localhost:9090/census/1998/communes?page=1&per_page=100
